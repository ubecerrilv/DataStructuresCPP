# Data Structures in C++

This repository contains a collection of **data structures exercises and examples implemented in C++**. The main goal is to strengthen fundamental programming concepts and improve problem-solving skills using classic data structures.

The exercises are intended for **academic learning and personal practice**, covering basic to intermediate data structures.

---

## Contents

This repository includes implementations and exercises related to:

- Arrays
- Linked Lists  
  - Singly linked lists  
  - Doubly linked lists
- Stacks
- Queues
- Trees  
  - Binary trees  
  - Binary search trees
- Graphs (basic concepts)
- Hash tables (as the repository grows)
- Basic algorithms related to data structures

> The content will continue to grow as new exercises are added.

---

## Requirements

To compile and run the programs, you will need:

- A compiler compatible with **C++11 or later**  
  - Example: `g++`
- Linux, Windows, or macOS
